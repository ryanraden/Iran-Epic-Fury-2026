Live Data
                Architecture

              

                GDELT (Auto) — DOC 2.0 API queries fire on page load
                and on the selected refresh interval (default: 1 hour). 5 parallel queries cover: Iran war volume, Iran
                tone, Hormuz coverage, nuclear escalation volume, and oil crisis volume. Charts update in place. Status
                shown in Data Feeds bar.


                World Monitor (Link) — No public API; live map opens
                via direct link with all layers enabled. Layer data is maintained in UPDATE BLOCKS in the
                source.


                Fallback: If GDELT API is unreachable (CORS block,
                outage, or offline use), all charts retain the last known static dataset from March 12, 2026. Feed dot
                turns red.
